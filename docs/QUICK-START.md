# 🚀 Быстрый старт - За 5 минут!

## Что вам нужно сделать ПРЯМО СЕЙЧАС

### Шаг 1: Запустить N8N (программа для визуализации)

Откройте терминал (черное окно) и введите:

```bash
npx n8n
```

**Что произойдет:**
```
✓ Загрузится N8N
✓ Запустится сервер
✓ Откроется браузер автоматически
✓ Вы увидите красивый интерфейс!
```

**Адрес в браузере:** http://localhost:5678

## 🎯 Простой пример (делайте вместе со мной!)

### Создадим: "Telegram уведомление по расписанию"

#### 1. Добавьте узел "Schedule Trigger"

Нажмите **[+]** в центре → Найдите **"Schedule"** → Выберите **"Schedule Trigger"**

```
┌─────────────────┐
│  Schedule       │ ← Этот кубик появится
│  Trigger        │
└─────────────────┘
```

**Настройте:**
- Mode: Every Day
- Hour: 9
- Minute: 0

**Означает:** Каждый день в 9:00 утра

---

#### 2. Добавьте узел "Telegram"

Нажмите **+** справа от Schedule → Найдите **"Telegram"** → Выберите

```
┌─────────────────┐      ┌─────────────────┐
│  Schedule       │ ───▶ │  Telegram       │
│  Trigger        │      │                 │
└─────────────────┘      └─────────────────┘
```

**Настройте:**
1. Credentials: Нажмите "+ Add Credential"
2. Введите Token от BotFather (получите из Telegram)
3. Chat ID: Ваш ID (можно узнать у @userinfobot в Telegram)
4. Text: "Доброе утро! Время работать! 🌅"

---

#### 3. Нажмите "Execute Workflow"

```
┌─────────────────┐      ┌─────────────────┐
│  Schedule       │ ──✓─▶│  Telegram       │
│  ✓ Выполнено    │      │  ✓ Отправлено   │
└─────────────────┘      └─────────────────┘
```

**Зеленая галочка = работает!** ✨

## 🔑 API ключи - Откуда брать?

### Telegram Bot Token (БЕСПЛАТНО!)

**1. Откройте Telegram**
**2. Найдите:** `@BotFather`
**3. Отправьте:** `/newbot`
**4. Ответьте на вопросы:**
   - Как назвать бота? → Напишите: "My First Bot"
   - Username бота? → Напишите: "my_first_bot" (должен заканчиваться на bot)

**5. Получите TOKEN:**
```
Done! Congratulations on your new bot.
Token: 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

**6. Скопируйте этот TOKEN** - это ваш API ключ!

## ⚠️ Важные моменты для новичков

### 1. Локальный vs Облачный N8N

**Локальный (npx n8n):**
- ✅ Бесплатно
- ✅ Работает только когда компьютер включен
- ✅ Данные на вашем компьютере
- ❌ Если выключите компьютер - автоматизация остановится

**Облачный (n8n.cloud):**
- ✅ Работает 24/7
- ✅ Не зависит от вашего компьютера
- ❌ Платно (от $20/месяц)

**Для начала используйте локальный!**

---

### 2. API ключи - СЕКРЕТНЫЕ!

**НИКОГДА не показывайте API ключи:**
- ❌ В скриншотах
- ❌ В видео
- ❌ Другим людям
- ❌ В GitHub (если загружаете код)

**API ключ = пароль от вашего аккаунта!**

---

### 3. Бесплатные vs Платные сервисы

**Бесплатные (для начала):**
- ✅ Telegram Bot
- ✅ Google Sheets (до лимита)
- ✅ Email
- ✅ Webhooks
- ✅ HTTP запросы

**Платные:**
- 💳 OpenAI / ChatGPT (~$5-20/месяц)
- 💳 Twilio SMS (~$1 за 100 СМС)
- 💳 SendGrid Email (платный тариф)

## 🆘 Если что-то не работает

### Проблема: N8N не запускается
**Решение:**
```bash
# Обновите Node.js
brew upgrade node

# Попробуйте снова
npx n8n
```

### Проблема: Telegram не отправляет сообщения
**Решение:**
1. Проверьте TOKEN - он должен быть длинный
2. Проверьте Chat ID - это число
3. Сначала напишите боту в Telegram (нажмите Start)
