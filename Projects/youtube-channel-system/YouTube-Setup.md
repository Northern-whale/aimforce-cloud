## Phase 7: Monetization Checklist

### Requirements for YouTube Partner Program (YPP):
- [ ] 1,000 subscribers
- [ ] 4,000 watch hours in the past 12 months
- [ ] Follow YouTube's Community Guidelines & Terms
- [ ] No active strikes on channel
- [ ] 2-factor authentication enabled
- [ ] AdSense account linked

### Application Process:
1. Go to YouTube Studio ‚Üí Monetization
2. Review Partner Program terms ‚Üí Accept
3. Connect AdSense account (or create new)
4. Set up payment info in AdSense
5. Submit channel for review
6. Wait 1-4 weeks for approval

### AdSense Setup:
1. Create AdSense account: [google.com/adsense](https://www.google.com/adsense)
2. Enter payment details
3. Verify mailing address (postcard verification)
4. Link to YouTube channel
5. Set payment threshold ($100 minimum)

### Ad Settings (Once Approved):
- [ ] Enable display ads
- [ ] Enable overlay ads
- [ ] Enable skippable video ads
- [ ] Enable non-skippable ads (use sparingly - can hurt retention)
- [ ] Enable mid-roll ads (videos >8 minutes)

**Recommended Ad Settings:**
- Videos <5 min: Pre-roll only
- Videos 5-10 min: Pre-roll + 1 mid-roll
- Videos >10 min: Pre-roll + mid-rolls every 3-5 min

## Phase 8: Analytics Setup

### Key Metrics to Track:

**Dashboard Favorites:**
1. **Watch time** - Most important for algorithm
2. **Click-through rate (CTR)** - Thumbnail effectiveness (aim for 4-10%)
3. **Average view duration** - How long people watch (aim for >50%)
4. **Subscriber growth** - Net new subscribers
5. **Traffic sources** - Where viewers find you

**Custom Reports:**
- Create weekly report: Views, watch time, subscribers
- Compare videos: Which topics perform best?
- Audience retention graphs: Where do people drop off?

### YouTube Studio Mobile App
- Download for iOS/Android
- Enable notifications for:
  - New comments
  - Subscriber milestones
  - Performance alerts

## Phase 10: Legal & Compliance

### Music Licensing:
- **YouTube Audio Library** (free, built-in)
- **Epidemic Sound** ($15/month - best for creators)
- **Artlist** ($9.99/month - unlimited downloads)
- **Soundstripe** ($15/month)

**Never use:**
- Copyrighted music without license
- Music from Spotify, Apple Music, etc.
- "Royalty-free" from unverified sources

### Video Footage:
- Film your own B-roll
- Use stock footage (Pexels, Pixabay - free)
- Always get permission for customer/pet appearances
- Create simple release form

### Release Form Template:
